Npm scripts build tools

Uses NPM scripts to automate Babel Prettier and other great build tools

This build tool uses all these devDependencies to automate away all the repetive tasks of modern web development. Just use the command line to navigate to the project and run npm run watch:all to start the server and watch all your files for changes. Then make any changes and watch it appear live in the browser

  • autoprefixer
  • babel-cli
  • babel-preset-env
  • browser-sync
  • imagemin-cli
  • node-sass
  • npm-run-all
  • onchange
  • postcss-cli
  • prettier
  • stylelint
